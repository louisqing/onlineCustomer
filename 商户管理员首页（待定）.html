复制

<script>

    (function(d, w, c) {

      if (w[c]) return;

      var s = d.createElement("script");

      w[c] = function() {

        (w[c].z = w[c].z || []).push(arguments);

      };

      s.async = true;

      s.src = "https://res.letchat.net/plug/widget.js";

      if (d.head) d.head.appendChild(s);

    })(document, window, "_NEXCHAT");

    _NEXCHAT("ini",{ channelId : "sa0KZ3" });

    _NEXCHAT('setLanguage',{ type: 'en' });

  </script>

请复制上面的 JavaScript 代码，插入到网页的 </body> 标签之前，然后访问网页。

代码插入需要一定的技术能力，建议请教网页维护人员完成代码插入。
